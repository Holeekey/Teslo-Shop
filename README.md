# Teslo API

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archvio __.env.template__ y renombrar a __.env__
4. Establecer variables de entorno
5. Levantar base de datos

```
docker-compose up -d
```
6. Activar modo desarrollo
```
yarn start:dev
```
7. Ejecutar Seed 
```
http://localhost:3000/api/seed
```
